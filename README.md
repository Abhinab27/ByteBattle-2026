# ByteBattle 2026

ByteBattle 2026 is a national-level coding challenge hosted by NIT Silchar, bringing together students from across India to solve real-world problems in Artificial Intelligence, Machine Learning, and Software Development.

## Features
- Event information and timeline
- Competition rounds and rules
- Prize and recognition details
- About NIT Silchar
- Contact and social media links
- Responsive modern UI

## Tech Stack
- HTML
- CSS
- JavaScript
- nextjs
- Tailwind CSS 

## About
ByteBattle 2026 serves as a platform for innovation, collaboration, and impactful problem-solving, providing participants with opportunities to work on industry-aligned challenges and emerging technologies.
