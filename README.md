# ByteBattle 2026

🔗 **Live Demo:** https://bytebattle-delta.vercel.app

ByteBattle 2026 is a national-level coding challenge hosted by NIT Silchar, bringing together students from across India to solve real-world problems in technology, innovation, and software development.

## Overview

This project is the official event website for ByteBattle 2026. The website provides participants with information about the competition, event timeline, rules, prizes, and details about NIT Silchar. The primary focus of this project was building a modern, responsive, and user-friendly frontend experience.

## Features

* Responsive design for desktop, tablet, and mobile devices
* Modern landing page with event highlights
* Competition details and event timeline
* Prize and recognition information
* About NIT Silchar section
* Contact and social media integration
* Fast and optimized user experience

## Tech Stack
